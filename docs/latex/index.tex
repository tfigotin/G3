\hypertarget{index_overview_sec}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{index_overview_sec}
Serendipity Booksellers is a small bookstore located in a shopping mall. The manager requested a point-\/of-\/sale (POS) software package that makes the computer function as both a cash register and an inventory database.

This project is developed incrementally across multiple chapters, eventually forming a complete, multi-\/module C++ application.

The software performs the following major tasks\+:
\begin{DoxyItemize}
\item Calculate the total of a sale, including sales tax
\item Subtract purchased books from the inventory database
\item Add, change, delete, and look up books in inventory
\item Display various inventory and sales reports
\end{DoxyItemize}\hypertarget{index_modules_sec}{}\doxysection{\texorpdfstring{Program Modules}{Program Modules}}\label{index_modules_sec}
The program is organized into three main modules\+:


\begin{DoxyItemize}
\item {\bfseries{Cashier Module}} ~\newline
 Acts as a cash register\+: collects purchases, computes totals and tax, and deducts quantities from the inventory.
\item {\bfseries{Inventory Database Module}} ~\newline
 Stores and manages all book records. Supports\+:
\begin{DoxyItemize}
\item Lookup (partial title or ISBN) ~\newline

\item Add new books ~\newline

\item Edit existing book data ~\newline

\item Delete books (with compaction) ~\newline

\item Display individual book information
\end{DoxyItemize}
\item {\bfseries{Report Module}} ~\newline
 Generates the following reports\+:
\begin{DoxyItemize}
\item Complete inventory list ~\newline

\item Wholesale value report ~\newline

\item Retail value report ~\newline

\item Books sorted by quantity ~\newline

\item Books sorted by wholesale cost ~\newline

\item Books sorted by date added ~\newline

\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{index_data_sec}{}\doxysection{\texorpdfstring{Book Record Structure}{Book Record Structure}}\label{index_data_sec}
Each book in the inventory stores the following fields\+:
\begin{DoxyItemize}
\item ISBN (string) ~\newline

\item Title ~\newline

\item Author ~\newline

\item Publisher ~\newline

\item Date Added ~\newline

\item Quantity on Hand ~\newline

\item Wholesale Cost ~\newline

\item Retail Price ~\newline
